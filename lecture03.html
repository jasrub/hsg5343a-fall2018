Semantic Elements and Structure

Why Use Semantic Tags:

  • accessibility - help assistive technologies read and interpret your webpage
  • searchability - help computers make sense of your content
  • internationalization
  • interoperability - help other programmers understand the structure of your webpage

Git is not Github

Git

Distributed version Control Software, used to keep track of changes within a project by taking snapshots of files over time

Github

Online service that allows you to upload and share your git repositories. github.com

How Git Works

  • Each project is stored in a Repository
  • The state is saved with every commit we make to the repository
  • Each commit is represented by a long string of letters and numbers called a hash

Github.com

  • Web based hosting platform for Git repositories
  • Very important site for sharing and reviewing code
  • Nice GUIs available: https://desktop.github.com/

Git "Areas"

Working Directory

Contains the current state of your files

Staging Area

Contains all the changes that will be included in your next commit

Git Repository
  • Where your whole project is stored
  • Can contain work from many different people
  • Contains all the commit objects from the life of the project
