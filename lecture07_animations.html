Transitions

Transitions

CSS Transitions allow us to gradually transition an element

from one state to another

Transition

Rules Definition

                    
                        transition-property: width;
                        transition-duration: 2s;
                        transition-timing-function: linear;

or:

                    
                        transition:  width  2s  linear;

https://codepen.io/jasrub/pen/qJyRwr

Transition

HTML

                    
                        <div class="box"></div>

CSS

                    
                        div.box {
                            transition: width  2s  linear;
                            width: 100px;
                            Background-color: red;
                            Height: 100px;
                        }

                        /* Transition Trigger */
                        div.box:hover {
	                       width: 200px;
                        }

Transitions

Timings

                    
                        transition-timing-function: linear;
                        transition-timing-function: ease-in;
                        transition-timing-function: ease-out;
                        transition-timing-function: ease-in-out;
                        transition-timing-function: cubic-bezier(…);

Transforms

                          You can transform this div by hovering over it!
                    
                        
                        #transform-example {
                          transform: rotateZ(5deg);
                          transition: transform 2s ease-in-out;
                        }

                        #transform-example:hover {
                          transform: rotateZ(-5deg);
                        }

Transforms

Other Transform Values

  • translate – translate element along X, Y, or Z axes (in deg, rad, turn, or %)
  • scale – scale size of element along X or Y axes (in integers – e.g. 2.0 for 200% scale)
  • rotate – rotate element along X, Y, or Z axes (in deg, rad, or turn)
  • skew – skew element along X or Y axes (in deg or rad)
  • perspective - set perspective from which element is viewed (in px)
  • Additional Properties

Animations

Animations

CSS Animations allow us to gradually transition an element

between multiple states

Animations

CSS Animations are defined in two steps (two separate rules)

  1. Define the animation
  2. Set the element to use the animation

Animations

Step 1 - Define the animation

                    
                    @keyframes slide {
                        from { 
                            background: red; 
                            left: 0; 
                        }
                        to { 
                            background: green; 
                            left: 100px; 
                        }
                    }

Animations

Step 2 - Set the element to use the animation

                    
                    .box {
                        position:relative;
                        height:100px;
                        width:100px;
                        /* define animation */
                        animation-name: slide;
                        animation-duration: 5s;
                        animation-iteration-count: 2;
                    }

                    /* in one line: */
                    .box {
	                   animation: slide   5s   2;
                    }

Animation

                    
                    @keyframes pulse {

                            from {
                             opacity: 0.0;
                             font-size: 100%;
                            }

                            to {
                             opacity: 1.0;
                             font-size: 200%;
                            }

                    }
                    div {
                      animation-name: pulse;
                      animation-duration: 2s;
                      animation-iteration-count: infinite;
                      animation-timing-function: ease-in;
                      animation-direction: alternate;
                    }

pulse

Animations

Multiple steps animation

                    
                    @keyframes widen {
                        0% { width: 100px; }
                        50% { width: 200px; }
                        100% { width: 100px;}
                    }

Multiple steps animation

                    
                    @keyframes rainbow {
                        0%   {
                          background-color: red;
                          width:500px;
                        }
                        25%  {
                          background-color: yellow;
                          width: 200px;
                        }
                        50%  {
                          background-color: blue;
                          width: 300px
                        }
                        100% {
                          background-color: green;
                          width: 200px;
                        }
                    }


                    .rainbow {
                      animation-name: rainbow;
                      animation-duration: 6s;
                      animation-iteration-count: infinite;
                      animation-timing-function: ease-in;
                      animation-direction: alternate;
                    }
rainbow

Animations

Animations parameters (set on the element CSS rule)

animation-direction Start playing from the first frame or last frame
animation-play-state whether the animation is running or paused
animation-fill-mode specifies a style for the element when the animation is not playing (before it starts, after it ends, or both)
animation-delay delay for the start of an animation
