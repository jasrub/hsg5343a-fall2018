HTML

Hyper Text Markup Language

  • Creates the structure of a web page
  • Gives Semantic meaning to the content

CSS

Cascading Style Sheets

  • Adds visual style to a page
  • Colors, fonts, borders etc.
  • Responsive control, various media
  • Adjectives of a web page

                            #button {
                              color: #ffffff;
                              background-color: purple;
                              font-family:'Helvetica';
                              font-weight:100;
                              font-size:20pt;
                            }

Javascript

  • Adds logic and interactivity to a page
  • Actions or verbs of a page

                            var b = document.getElementById("button");
                            b.onclick = function() {alert("Thanks for clicking!")};
