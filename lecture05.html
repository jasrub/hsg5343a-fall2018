Styling HTML with CSS

CSS can be added to HTML elements in 3 ways:

  • External - by using an external CSS file. declare in <head> section [preferred] 
    <link rel="stylesheet" href="styles.css">
  • Internal - by using a <style> element in the <head> section [meh] 
    
                                <style>
                                    body {background-color: powderblue;}
                                    h1   {color: blue;}
                                    p    {color: red;}
                                </style>
  • Inline - by using the style attribute in HTML elements [yuck] 
    <h1 style="color:blue;">This is a Blue Heading</h1>

Box Model

All HTML elements can be considered as boxes. In CSS, the term "box model" is used when talking about design and layout. The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.

Box Model

  • content - The content box where text and images appear
  • padding - Clears an area around the content. The padding is transparent
  • border - A border that goes around the padding and content
  • margin - Clears and area around the border. The padding is transparent

Box Model

To the width and height of our element, the border and margins are added.
Padding is added internally to the box object and so doesn't add to the height and width

Box Model - Content

Height and Width of the contet area are determined by the display attribute, the size of it's content (image/text) and the height and width attributes.

Box Model - Padding

Padding is set by the css attribute padding. It creates internal spacing around the content.The padding is transparent and can't have it's own color.

Box Model - Padding


                   padding: 20px; /* uniform spacing around the element */
                

                    padding-top: 5px;
                    padding-right: 15px;
                    padding-bottom: 20px;
                    padding-left: 25px;
                

                   padding: 5px 15px 20px 25px;  /* Top   Right   Bottom   Left */
                

                   padding: 5px 20px;  /* Top Bottom    Right Left */

Box Model - Border

A border that goes around the padding and content

Box Model - Border


                    border-style: solid;
                    border-width: 15px;
                    border-radius: 25px;
                    border-color: #00ff00;
                

                   border: 1px solid #00ff00;  /* Width    style     color  */

Box Model - Margin

Sets the spaces on the outside of the box, between the element and other elements on the page or the page edges.

Box Model - Margin


                   margin: 20px; /* uniform spacing around the element */
                

                    margin-top: 5px;
                    margin-right: 15px;
                    margin-bottom: 20px;
                    margin-left: 25px;
                

                   margin: 5px 15px 20px 25px;  /* Top   Right   Bottom   Left */
                

                   margin: 5px 20px;  /* Top Bottom    Right Left */

CSS- Height and Width

  • The width and height of an element can be set in pixels or percentages
  • If they are set in percent, the width and height of the element will be relative to the width and height of the element that contains them in the parent html
  • The default size of an element will be auto - it means it will be defined by its content - text amount or image size
  • In case an element (tag) has no content (and its children ...) if it has no width settings and its size is 0px / 0px
