Styling HTML with CSS

CSS can be added to HTML elements in 3 ways:

  • External - by using an external CSS file. declare in <head> section [preferred] 
    <link rel="stylesheet" href="styles.css">
  • Internal - by using a <style> element in the <head> section [meh] 
    
                                <style>
                                    body {background-color: powderblue;}
                                    h1   {color: blue;}
                                    p    {color: red;}
                                </style>
  • Inline - by using the style attribute in HTML elements [yuck] 
    <h1 style="color:blue;">This is a Blue Heading</h1>
